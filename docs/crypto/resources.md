# Resources/Tools

The following is a hastily put-together list of relevant tools and resources.

- [Crypto Attacks](https://github.com/jvdsn/crypto-attacks) - various cryptographic attacks in python
- [yafu](https://github.com/bbuhrow/yafu) - Automated integer factorization
- [Ciphey](https://github.com/Ciphey/Ciphey) - Automated decryptor/brute-forcer (good for low-hanging fruit)
- [quipquip](https://quipqiup.com/) - helpful for solving more traditional ciphers
- [Morse code decoder](https://github.com/Razer2015/MorseDecoder)
- [bitwise operations](https://nicolwk.medium.com/bitwise-operations-cheat-sheet-743e09aec5b5) - silly cheatsheet
- [Hash Length Extension Attack](https://book.hacktricks.xyz/crypto-and-stego/hash-length-extension-attack)
- [Timing attack talks](https://grhkm21.github.io/posts/timing-attack-talk/)
